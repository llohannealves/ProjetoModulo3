## 📌 Desafio Escolhido

Descreva aqui o desafio proposto pelo grupo (ex.: cadastro de usuários, automação de envio de e-mails, integração com API de IA).

O desafio escolhido pelo grupo foi a integração com API de IA. Nosso protótipo tem a ideia de reunir pessoas com as melhores ofertas para ela. Funciona como se fosse um garimpo on-line onde o usuário pode fazer uma breve pesquisa de qual peça ele irá querer, e o app fornece uma busca fornecendo as melhores opções para atender o usuário com o melhor custo beneficio. O app fornece também, o chat com profissionais onde o usuário pode entrar em contato e conversar diretamente sobre sua buscar e auxilia-lo em conseguir a melhor opção.
